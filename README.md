## Para Iniciar

- Primero, Instala las dependencias:

```bash
npm install
# or
yarn install

```

- Segundo, Levanta el servidor:

```bash
npm run dev
# or
yarn dev
```

Abre [http://localhost:3000](http://localhost:3000) con tu navegador de confianza.
